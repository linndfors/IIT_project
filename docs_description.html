lsb_stego.decode_lsb(file_path)[source]¶

Витягує та декодує приховане повідомлення із захищеного WAV-файлу.

Функція зчитує LSB (найменші значущі біти) з кожного байту аудіоданих, формує з них символи та шукає маркер закінчення („#####END“).

Параметри:

file_path (str) – Шлях до файлу (.wav), який потрібно перевірити.

Повертає:

Розшифрований рядок повідомлення (без маркера), якщо його знайдено. Повертає None, якщо маркер не знайдено або файл не містить прихованих даних.

Тип повернення:

str | None

Викликає:

Exception – При помилках читання файлу.

lsb_stego.encode_lsb(input_path, output_path, secret_message)[source]¶

Вбудовує приховане текстове повідомлення у аудіофайл формату WAV методом LSB.

Функція конвертує рядок повідомлення у бінарну послідовність, додає спеціальний маркер закінчення („#####END“) та замінює найменші значущі біти (Least Significant Bits) аудіоданих на біти повідомлення.

Параметри:

  • input_path (str) – Шлях до вхідного незахищеного файлу (.wav).

  • output_path (str) – Шлях, куди буде збережено захищений файл.

  • secret_message (str) – Унікальний рядок (Payload), який потрібно сховати (наприклад, ISRC або ID власника).

Повертає:

True, якщо вбудовування пройшло успішно, інакше False.

Тип повернення:

bool

Викликає:

  • ValueError – Якщо розмір аудіофайлу занадто малий для вміщення повідомлення.

  • Exception – При помилках відкриття файлу або запису.

app.protect()[source]¶

Основний маршрут для захисту аудіофайлу.

Алгоритм роботи: 1. Приймає файл (.wav або .mp3) та метадані. 2. Якщо файл у форматі MP3, виконує конвертацію у WAV (використовуючи pydub/ffmpeg). 3. Генерує унікальний ідентифікатор (UUID). 4. Викликає модуль lsb_stego для вбудовування ідентифікатора у біти файлу. 5. Зберігає інформацію в БД та генерує PDF-сертифікат.

Повертає:

Рендер сторінки або перенаправлення на dashboard.

app.verify()[source]¶

Маршрут для аудиту (перевірки) файлів.

Дозволяє завантажити підозрілий файл, спробувати декодувати LSB та перевірити наявність витягнутого ключа в базі даних. Повертає статус PROTECTED (із даними власника) або CLEAN.
